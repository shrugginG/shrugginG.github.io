什么是Github

2020-11-18 Comments

基本概念

什么是Git？

什么是版本控制系统？

Git是与版本控制系统的概念是息息相关的，我们每次对文件进行修改都相当于文件迭代出了一个新版本，而版本控制系统就可以追踪我们的更新并为我们记录下了每次版本的更新。有了版本追踪系统我们就可以实现版本的回滚。

了解了什么是版本控制系统那么到底什么是Git呢？

简单来说，Git就是一款免费的、开源的分布式控制系统。

相当于本地、公司服务器、Github网站服务器都装Git做版本控制，只不过Github的服务器强大些，对全球用户托管的项目用Git做版本控制。
正是由于Github用Git做版本控制，所以可以轻松的记录项目的变迁史。

什么是Github？

GitHub 是一个面向开源及私有软件项目的托管平台，因为只支持 Git 作为唯一的版本库格式进行托管，故名 GitHub。

Git中的几个概念

仓库(Repository)

每一个仓库都对应着一个开源项目。

收藏(Star)

仓库主页Star按钮，收藏之后方便下次快速查找。

复制克隆项目(Fork)

这个也就是通常我们所说的把别人的项目“叉下来”，这样在我们的个人中心就多了一份跟原项目一模一样的仓库。但是这里特别注意我们“叉下来”的项目是独立存在的，并不会影响到别人的原项目。

发起请求(Pull Request)

我们可以在fork完别人的项目以后做一些自己的改动然后通过Request发送给原作者查看，如果原作者采纳就可以合并到原仓库中。

关注(Watch)

就是类似于社交网站上的关注，但是关注的是项目，当项目有更新时，会第一时间提示。

事务卡片(Issue)

当你发现了项目中存在bug等一类问题就可以提出一个issue以供大家讨论。

GitHub主页

账号注册成功以后点击网址导航栏的github图标就可以进入GitHub的主页，该页左侧主要显示用户的动态以及用户关注仓库的动态，右侧显示所有的Git库。

仓库主页

仓库主页主要显示项目的具体信息。如：项目代码，版本，收藏，关注，fork等情况。

个人主页

主要显示个人信息，头像，个人的简介，关注我的人，我关注的Git库，我的开源项目，以及我贡献的开源项目等信息。

创建仓库(Repository)

一个开源项目就对应着一个仓库。
